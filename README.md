# Beacon Design System

A guiding light for exploration through design.

## Development

This repository is a Turborepo monorepo that contains the following projects:

- packages/
  - react
- apps/
  - [website](./apps/website/README.md)

Prerequisites:

- Node.js
- npm

```bash
# install dependencies
npm i

# build all packages
npm run build
```
